Administración de Mercados Públicos

El sistema Mercaditosh permite administrar información sobre mercados públicos en la Ciudad de México, incluyendo gestión de mercados, locales comerciales, locatarios, productos, pagos de renta e inspecciones sanitarias.

Este proyecto forma parte de la materia Tecnologías para el Desarrollo de Aplicaciones Web en ESCOM - IPN.

Módulos del Sistema

Nuevo

Mercados

Registro de mercados públicos con nombre, dirección y alcaldía.

Locales

Administración de locales comerciales dentro de cada mercado.

Locatarios

Registro de comerciantes que ocupan los locales.

Pagos

Seguimiento de pagos de renta realizados por los locatarios.

Inspecciones

Registro de inspecciones sanitarias realizadas a los locales.

Plataforma Web Responsive

La plataforma Mercaditosh está diseñada para adaptarse a diferentes dispositivos gracias al uso de Media Queries y Flexbox. Esto permite que el sistema pueda utilizarse en computadoras, tablets y teléfonos móviles sin perder funcionalidad.
